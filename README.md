## About the Project

The **Travel World** is a full-stack web application built using **React** for the frontend and **Firebase** for backend services like authentication and hosting. The app allows users to:

- Browse various travel destinations
- Book tours and check availability
- View detailed information about each destination
- Sign up, log in, and manage bookings

The app is designed to be responsive, providing an excellent user experience on all devices.

---

## Features

- **Homepage:** Displays featured destinations and a hero section.
- **Destination Listings:** Browse destinations with detailed information about each.
- **Booking System:** Book tours by selecting dates and number of people.
- **User Authentication:** Sign up, log in, and manage bookings.
- **Responsive Design:** Fully responsive, ensuring a seamless experience across devices.

## Installation

### Prerequisites

Ensure that **Node.js** and **npm** are installed on your machine.

### Steps to Run Locally

1. Clone the repository:
   ```bash
   git clone https://github.com/aniruddhasahaa/Tour-Booking-Project.git
   ```

2. Navigate into the project directory:
   ```bash
   cd Tour-Booking-Project
   ```

3. Install dependencies:
   ```bash
   npm install
   ```

4. Run the development server:
   ```bash
   npm start
   ```

   The app will be available at `http://localhost:3000`.

## Usage

1. Visit the homepage to see featured destinations.
2. Explore different sections of the site via the navigation bar.
3. Sign up or log in to make bookings and view your booking history.
4. Navigate to the booking page, select your destination, dates, and book your tour.
